# 📖 Bookstore <a name="about-project"></a>


**Bookstore** is a The Bookstore is a simplified website built with React and Redux. It allows users to view and manage a list of books, including adding and removing books.

## 💻 Getting Started <a name="getting-started"></a>

To get a local copy up and running, follow these steps:

- Clone this project.
- Open it in VS code.
- Run the following commands to install the dependencies
  - npm install
- Deploy it on the dev-server using npm start.

### Prerequisites

In order to run this project you need:

Git A GitHub account VS-Code React and Redux linters Node.js

### Setup

Clone this repository to your desired folder:

https://github.com/Sonikak004/book-store.git


### Deployment

You can deploy this project through the dev server. To initiate it, run npm start in your cli and the project will open a dev server at http://localhost:3000/
